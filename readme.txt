Querlo Chatbot plugin lets you install a free Querlo chat directly on your WordPress site page.

== Description ==
Querlo Chatbot plugin lets you install a free Querlo chat directly on your WordPress site pages. With one click, your website visitors can have a conversation with Querlo virtual assistant. A native Querlo chat button is easily recognizable by anyone and encourages trustful communication with all the features that a Querlo Customer Chat can offer.

What makes Querlo Chatbot plugin stand out?
* Easy chat setup, no programming skills needed. Pre-built templates for several use cases are available.
* Direct communication with users on the page, without leaving the site.
* Instant receiving and replying to messages so your customers arenâ€™t lost.
* Chatbot can be used to automate responses to customer questions and reduce the amount of support agents.

== Installation ==
1. Upload the plugin to your FTP or other directory, then install it through the Plugins menu in your WordPress Dashboard.
2. To use the Querlo Customer Chat, you need a Querlo account. Lifetime free plan available. Please create one at https://www.querlo.com/register
